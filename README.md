# 基于koa的后端快速启动框架

## 技术栈

|   ··   |  技术   |       node 库       |
| :----: | :-----: | :-----------------: |
|  框架  |   ts    |         koa         |
|  路由  |   --    |     koa-router      |
| 数据库 | mongoDB |      mongoose       |
|  缓存  |  redis  |       ioredis       |
|  时间  |   --    |       moment        |
|  语言  |   --    |     typescript      |
| script |   --    | nodemon/ coress-env |


## features

1. 路由
   1. 支持频率限制(需要redis)
   2. 用户登录判断
   3. 接口字段判断
   4. 多路由情况下重复路由判断
2. 日志
   1. 格式化输出
   2. 自定义格式输出
3. ......(敬请期待)
